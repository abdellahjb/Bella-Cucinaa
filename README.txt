Bella Cucina – Site Web de Restaurant

Description

Bella Cucina est un site web moderne et élégant conçu pour présenter un
restaurant de cuisine italienne haut de gamme. Le site comprend
plusieurs sections : accueil, menu, à propos, galerie et contact. Il est
construit avec HTML, CSS et Bootstrap 5, en utilisant Font Awesome pour
les icônes et Google Fonts pour la typographie.

Fonctionnalités

-   Barre de navigation responsive
-   Section d’accueil (Hero) avec le nom du restaurant et un slogan
-   Section Menu avec plusieurs catégories (Entrées, Salades, Pizzas,
    Pâtes, Plats principaux, Desserts)
-   Section À propos présentant l’histoire et les valeurs du restaurant
-   Galerie d’images avec effets au survol
-   Section Contact avec les informations du restaurant et un formulaire
    de contact
-   Design moderne basé sur Bootstrap 5 et du CSS personnalisé
-   Icônes fournies par Font Awesome
-   Polices depuis Google Fonts (Playfair Display & Poppins)

Technologies Utilisées

-   HTML5
-   CSS3
-   Bootstrap 5.3
-   Font Awesome 6
-   Google Fonts

Structure des Fichiers

-   index.html -> Fichier principal du site (contient toutes les
    sections)
-   README.txt -> Ce fichier (description du projet et instructions)

Comment Utiliser

1.  Téléchargez ou copiez les fichiers du projet.
2.  Ouvrez le fichier index.html dans un navigateur moderne (Chrome,
    Edge, Firefox, etc.).
3.  Le site se lance automatiquement, aucun serveur ou installation
    n’est nécessaire.

Personnalisation

-   Vous pouvez changer le nom du restaurant, les textes et les prix
    directement dans index.html.
-   Vous pouvez remplacer les images en modifiant les URLs dans les
    balises <img>.
-   Les couleurs, polices et styles peuvent être modifiés dans la
    section <style> du fichier HTML.
-   Mettez à jour les informations de contact (adresse, téléphone,
    email, horaires) dans la section Contact.
